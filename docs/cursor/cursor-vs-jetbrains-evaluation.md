## Quick Reference: Your Options

### Option A: Cursor Pro + IntelliJ IDEA
- **Cost**: $240/year (Cursor Pro) + $149/year (IntelliJ) = **$389/year**
- **Use case**: Cursor for Python/JS greenfield, IntelliJ for Scala
- **Bet**: AI-first tools are worth the premium for learning new languages

### Option B: JetBrains All Products Pack
- **Cost**: $179/year
- **Use case**: IntelliJ/PyCharm for all work, Claude Code as separate tool
- **Bet**: Traditional IDE depth > AI assistance for now

### Option C: IntelliJ IDEA Only
- **Cost**: $149/year
- **Use case**: IntelliJ for Scala, free tools (VS Code + extensions) for Python/JS
- **Bet**: Don't need paid tools for Python/JS
