# Notes on Ticker Symbol Resolution

## NASDAQ

| Index Name        | Ticker   | Coverage                         | Tech Weighting           |
| ----------------- | -------- | -------------------------------- | ------------------------ |
| Nasdaq Composite  | ^IXIC    | Nearly all Nasdaq stocks         | Extremely high           |
| Nasdaq-100        | ^NDX/QQQ | 100 largest non-financial stocks | Highest (large-cap tech) |
| US Tech 100 Index | Various  | Synonym for Nasdaq-100           | Highest                  |

Hedgeye Risk Range Reports refer to the Nasdaq Composite using the symbol `COMPQ`.
Though when taking a position in the Nasdaq, Hedgeye buys `QQQ`, which is actually based on the Nasdaq-100.
