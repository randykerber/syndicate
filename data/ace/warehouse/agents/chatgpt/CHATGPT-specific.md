## Creekview Flora & Fauna Project

### Purpose
Build a catalog of species on property (wildlife-rich environment).

### Guidelines
- **Always include scientific names** for species
- **Unique species entries only** (no duplicates)
- **Track pest vs beneficial species**
- Document observations with context (behavior, habitat, season)

### Species Documented
- Grasshopper log (various species)
- Red velvet ant
- Wasps (multiple varieties)
- Lizards (species TBD)
- Turkey flock with chicks
- Various flora (trees, shrubs, wildflowers)

### Data Structure
Prefer structured format:
```markdown
## Species Name (Scientific Name)
- **Type**: Flora/Fauna
- **Classification**: Beneficial/Pest/Neutral
- **Observed**: Date, location on property
- **Behavior**: Key observations
- **Notes**: Additional context
```
